    # 关于GB/T 28181

    GB/T 28181 全称为《公共安全视频监控联网系统信息传输、交换、控制技术要求》，是我国视频监控领域最重要的国家级通信标准。

    该标准由中华人民共和国公安部提出。 该标准由全国安全防范报警系统标准化技术委员会(SAC/TC100)归口。该标准起草单位:公安部第一研究所、视频图像信息智能分析与共享应用技术国家工程实验室、国家信息中心、杭州海康威视数字技术股份有限公司、浙江大华技术股份有限公司、浙江宇视科技有限公司、华为技术有限公司、中星微技术股份有限公司。

    # 标准定位

    该标准的核心目标是解决视频监控系统中不同厂商设备无法互联、不同平台之间视频资源难以共享的问题，为公共安全视频监控联网系统提供统一的信令控制和媒体传输规范，是实现视频监控系统互联互通的基础。

    适用于公共安全视频监控联网系统的方案设计、系统检测、验收，以及与之相关的设备研发和生产。其他视频监控联网系统也可参照执行。

    # 三大国际信息通信标准组织简介

    在全球信息通信（ICT）领域，最具影响力的三大标准化组织之一分别是：

    • 国际电信联盟open in new window（ITU，尤其是 ITU-T）
    • 国际标准化组织open in new window（ISO）
    • 互联网工程任务组open in new window（IETF）

    它们共同推动了互联网、通信网络、视频监控、人工智能、物联网等领域的标准化发展。

    # 1. ITU-T（国际电信联盟电信标准化部门）

    # 基本信息

    • 全称：International Telecommunication Union - Telecommunication Standardization Sector
    • 中文：国际电信联盟电信标准化部门
    • 隶属于：国际电信联盟
    • 成立时间：1865年（ITU前身）
    • 总部：日内瓦
    • 联合国专门机构

    # 主要职责

    负责全球电信网络和通信技术标准制定：

    • 固定电话网络
    • 移动通信网络
    • 光纤通信
    • 视频编码
    • 视频会议
    • IPTV
    • 网络安全
    • AI治理

    # 著名标准

    标准说明
    H.264视频压缩编码
    H.265 (HEVC)超高清视频编码
    H.266 (VVC)下一代视频编码
    G.711PCM语音编码
    G.729语音压缩
    X.509数字证书体系
    Y.3172AI网络框架

    # 与视频监控行业关系

    视频监控领域大量采用：

    • H.264
    • H.265
    • RTP
    • RTSP
    • QoS标准

    海康、大华、宇视等厂商设备均遵循相关ITU-T视频编码标准。

    # 2. ISO（国际标准化组织）

    # 基本信息

    • 全称：International Organization for Standardization
    • 中文：国际标准化组织
    • 成立时间：1947年
    • 总部：日内瓦
    • 成员国家：170多个

    # 主要职责

    制定全球通用标准：

    覆盖：

    • 工业制造
    • 软件工程
    • 信息安全
    • 人工智能
    • 质量管理
    • 医疗设备
    • 环境管理

    # 著名标准

    标准领域
    ISO 9001质量管理
    ISO 14001环境管理
    ISO/IEC 27001信息安全
    ISO/IEC 20000IT服务管理
    ISO/IEC 12207软件生命周期
    ISO/IEC 25010软件质量模型
    ISO/IEC 42001AI管理体系

    # 软件工程领域

    C++工程化实践常涉及：

    • ISO/IEC 12207
    • ISO/IEC 15504（SPICE）
    • ISO/IEC 25010

    这些标准规范：

    • 需求管理
    • 设计
    • 编码
    • 测试
    • 运维

    # 3. IETF（互联网工程任务组）

    # 基本信息

    • 全称：Internet Engineering Task Force
    • 中文：互联网工程任务组
    • 成立时间：1986年
    • 组织性质：开放性国际技术社区

    # 特点

    与ITU、ISO最大的不同：

    • 完全开放
    • 任何人都可参与
    • 以工程实践驱动
    • 强调“Running Code”

    著名理念：

    Rough Consensus and Running Code

    （粗略共识与可运行代码）

    # RFC机制

    IETF标准以RFC形式发布：

    RFC（Request For Comments）

    互联网世界几乎所有核心协议都来自RFC。

    # 在视频监控行业中的关系

                 ISO
              │
      软件质量/安全管理
              │

IETF ─────── ONVIF ─────── ITU-T
 │                           │
RTSP                       H.264
RTP                        H.265
HTTP                       视频编码
SIP                        QoS
TLS                        传输优化

    对于安防监控、视频会议、直播流媒体领域：

    • ITU-T 负责视频编码标准（H.264/H.265/H.266）
    • IETF 负责网络传输协议（RTP/RTSP/SIP/WebRTC）
    • ISO 负责软件工程、信息安全和质量体系标准

    三者共同构成了现代互联网视频产业的标准基础设施。
