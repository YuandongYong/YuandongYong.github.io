## WS-Discovery
- WS-Discovery（Web Services Dynamic Discovery）协议的官方技术规范由 OASIS（结构化信息标准促进组织）制定。
- [官方文档](http://docs.oasis-open.org/ws-dd/discovery/1.1/os/wsdd-discovery-1.1-spec-os.html)
- 该协议主要用于在本地网络中动态发现提供 Web Service 服务的设备，打破了传统服务调用中客户端需要预先知道目标服务地址的限制。

## ONVIF合规产品
- ONVIF 合规产品可以更轻松地集成不同供应商的产品，为系统集成商和终端用户提供了更大的灵活性和产品选择，并避免锁定单一供应商从而提供更简
  便的系统集成。迄今为止，市场上 拥有超过33,000种符合 ONVIF 标准的产品，并且安装了数百万台此类产品。
- ONVIF [Profile合规产品页面](https://www.onvif.org/ch/conformant-products/)是市场上判断产品及供应商是否符合正式ONVIF标准的。
